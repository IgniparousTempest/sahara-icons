# Contributing

## Adding a new icon

1. Add the SVG to `icons/svg/`
2. Add an entry to the Icons table in `README.md` in alphabetical order:
   ```
   | ![name](icons/svg/name.svg) | name | Human Readable Label |
   ```
3. Use [Conventional Commits](https://www.conventionalcommits.org/):
   ```
   feat: add <icon_name> icon
   ```
